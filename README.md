## üíé Overview
Vista streamlines complex formulation processes, manages multi-tier inventory, and provides real-time financial analytics for liquid-based product businesses (e.g., Skin Care and Hair Care).

## üöÄ Key Features

### ‚öóÔ∏è Precision Formula Engine
- **Dynamic Costing**: Automatically calculate finished unit costs based on real-time market prices of raw ingredients.
- **Complexity Tracking**: Monitor ingredient density and resource distribution for every SKU.
- **Margin Guard**: Real-time margin projections during the formula creation stage.

### üè≠ Intelligent Production Hub
- **Dual-Mode Production**: Execute runs by **Unit Count** (bottles) or **Total Batch Weight** (kg/L).
- **Auto-Deduction**: Real-time inventory sync that consumes raw ingredients and packaging upon completion.
- **Constraint Simulation**: Visual alerts for insufficient stock before a production run begins.

### üìä Financial Analytics & Inventory
- **Projected Profitability**: Calculate potential revenue based on current sellable stock.
- **Inventory Valuation**: Monitor capital tied up in raw materials vs. finished goods.
- **Market Price Sync**: Update market prices globally to reflect across all associated formulas.
